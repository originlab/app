Supporting Information

To open the Define Custom Design dialog

  • Activate a worksheet with your experimental design data
  • Select Statistics: Quality Improvement: Design of Experiment from Origin menu
  • Select the icon and choose the desired design from the list

Factors

The Column Long Names from the active worksheet are displayed in the list box. Select the desired factors and add them to the right panel.

Note: For Factorial Design you can select the design you want to define.

2-Level Factorial Define a 2-level factorial design from data in the worksheet.
General Full Factorial Define a general full factorial design from data in the worksheet.

Low/High

After defining the factors in the Factors tab, enter design information of factors in the Low/High tab

Type Factors can be text or numeric. If you want to treat a numeric column as text, right-click the column and select Set as Categorical from the context menu.
Low Display the minimum value of the numeric column or the first text of the categorial/text column by default. You can enter another desired value in it
High Display the maximum value of the numeric column or the second text of the categorial/text column by default. You can enter another desired value in it

Designs

Click the triangle button to specify which column in the active worksheet will contain the design information. If not specified, a new column will be created automatically in the result worksheet.

Run Order The order that the experimental runs should be done
Standard Order The standard order of the runs
Point Type The type of point
  • 0: Indicates center points
  • 1: Indicates factorial points

If no column of point type is specified, a column with all values set to 1 will be created automatically.

Blocks If no block column is specified, a column with all values set to 1 will be created automatically, indicating no blocking.


If keep all options to be <optional>

Response

After specifying factors in the Factors tab, the remaining columns will be shown in the available list in the Response tab. Select the desired response and add it to the right panel.
