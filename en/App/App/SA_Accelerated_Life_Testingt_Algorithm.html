2.5.4.2 Algorithm: Accelerated Life Testing

SA-Accelerated-Life-Testing-Algorithm

This tool supports two types of input.

Uncensor/Right Censor data: The data are represented as time and censoring indicator pairs :

  • : The observed time for each unit. This could be Exact failure time (if the unit failed) or Time to censoring (if the unit did not fail within the observation period)
  • : Indicates whether the unitis a failure or censored. e.g. 1 = failure (uncensored), 0 = censored

Uncensor/Arbitrary Censor data: The data are represented as time intervals :

  • : lower bound (time of last inspection or last known survival)
  • : upper bound (time when failure was first detected)
  • If , it represents right-censoring.
  • If , it represents exact failure (uncensored).

Probability Plot

Uncensor/Right Censor data

Points on the plot is calculated by the Benard's method.

Uncensor/Arbitrary Censor data

Points on the plot is calculated by the Turbull estimation[1]. Turnbull developed an iterative algorithm to obtain the nonparametric maximum likelihood estimate (NPMLE) of the cumulative distribution function for censored data. This approach is applicable to more general cases, including situations where the observation intervals overlap.
