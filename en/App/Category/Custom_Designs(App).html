2.43.1 Define Custom Designs

Custom-Designs

The Define Custom Design feature in the DOE (Design of Experiments) App allows users to create an experimental design from data already existing in a worksheet, rather than generating a new design from scratch. This is useful when you have a pre-defined set of experimental runs or want to import and analyze a design that was created externally.

Following design types are supported

  • Definitive Screening Design
  • Factorial Design
  • Response Surface Design
  • Mixture Design
  • Taguchi Design

Processing

  • Import Existing Data: Load your experimental design data into a worksheet.
  • Select the icon and choose desired design from the list
  • Specify settings in the opened dialog to create a design table
