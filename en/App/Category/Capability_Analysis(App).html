Introduction

Capability analysis can be used to assess whether a process is able to produce products that meet the specifications by providing the following quality tools. It is used to

  • Determine whether the process meets the specifications
  • How will the process perform in the future
  • Provide insight into how to improve the process and sustain your improvements.

Processing

Select a capability analysis based on your data

If there are multiple variables or multiple groups within one variable, three analysis are available

Tips: Specifications for Analysis for Multiple Variables

Input can be a number, space-seperated numbers or columns

Data Arrangement Description
A number
The value applies to all variables or groups
Space-seperated numbers
The values apply sequentially to the columns in Measurement Data or groups in Grouping Variable.
Columns

The values stored in column are applied sequentially to the columns in Measurement Data or groups in Grouping Variable.
