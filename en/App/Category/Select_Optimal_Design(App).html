2.43.2 Select Optimal Design

Select-Optimal-Design

Introduction

Select Optimal Design is based on D-optimality and can be used to select a fraction of experimental runs, thereby reducing the total number of trials. It can also augment an existing design by adding experimental runs if experimental resources are sufficient.

Processing

  • Prepare a worksheet containing your candidate set (or generate one via Generate Design).
  • Select Statistics: Quality Improvement: Design of Experiment from the Origin menu
  • Select the Select Optimal Design icon
